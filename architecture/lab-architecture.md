# Homelab Architecture

## Overview

This homelab environment is designed to support cybersecurity learning through hands-on practice with Linux servers, networking and security analysis.

## Current Structure

Host Machine
Windows 11 laptop running virtualization software.

Virtualization Layer
VirtualBox

Virtual Machines
Ubuntu Server (Cyber Lab)

Network Mode
NAT (default VirtualBox network)

## Network Details

Internal IP Address:
10.0.2.15

Service Enabled:
SSH (Secure Shell)

## Purpose of the Lab

Practice:
Linux administration
Networking fundamentals
System monitoring
Security analysis

## Planned Expansion

Add more virtual machines for:

* Attack simulation environment
* Security monitoring server
* Active Directory lab
* Logging and SIEM practice
